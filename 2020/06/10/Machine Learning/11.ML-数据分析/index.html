ML-数据分析

发表于 2020-06-10 分类于 Machine Learning
本文字数： 11k 阅读时长 ≈ 10 分钟

数据获取

  1. 数据仓库：
    • 主旨面向数据分析，方便不同维度抽取和整理，面向主题存储（谁在什么时候干了什么，多维度信息）。
  2. 检测与抓取：
    • 主要是爬虫
  3. 填写、日志、埋点：
    • 用户填写的信息
    • APP或网页埋点，特定流程的信息记录点，比如访客停留时间，页面查看和跳出率…主要分为两类：页面统计、操作行为。记录了用户与产品的交互过程。
    • 日志，分前端日志和后端日志
  4. 计算：
    • 某些指标不能直接获取，需要计算，比如适应率，投入产出比

数据分析

  • 集中趋势：均值、中位数、众数、分位数。
  • 离中趋势：标准差、方差。
  • 数据分布：偏态与峰态、正态分布与三大分布。
  • 抽样理论：抽样误差、抽样精度。

集中趋势：聚拢程度

  1. 均值：主要衡量连续值集中趋势，分布均匀，规律的连续值效果最好。
  2. 中位数：有异常值（离群点）时，衡量数据的集中趋势。
  3. 众数：主要衡量离散值的集中趋势。
  4. 分位数：共同作用，常用四分位数。
  5. 四分位数(n是数据总数)：
    • Q1的位置：(n+1)*0.25
    • Q2的位置：(n+1)*0.5
    • Q3的位置：(n+1)*0.75

离中趋势：离散程度

  • 方差：∑i=1n(xi−xˉ)2n\dfrac {\sum \limits_{i=1}^n (x_i - \bar x)^2} {n}ni=1∑n​(xi​−xˉ)2​、标准差：∑i=1n(xi−xˉ)2n\sqrt{\dfrac {\sum \limits_{i=1}^n (x_i - \bar x)^2} {n}}ni=1∑n​(xi​−xˉ)2​​

  • 方差和标准差的值，越大表示越离散，越小表示越聚拢。

  • 正态分布的离中趋势（σ是标准差，如果算出σ，且该特征符合正态分布）：
    * 数据落在-1σ~+1σ的概率是69%，
    * 数据落在-1.96σ~+1.96σ的概率是95%，
    * 数据落在-2.58σ~+2.58σ的概率是99%。

数据分布

  • 偏态系数：数据平均值偏离中位数状态的一种衡量。
    • 正常情况下，对称数据，数据的均值和中位数是接近甚至相等的。
    • 然而有的时候数据并非对称分布，其均值和中位数相差很大，这样的数据就是有偏态的分布。

  • 偏态系数公式：

    • S=1n∑i=1n(xi−xˉ)3[1n∑i=1n(xi−xˉ)2]3/2S = \dfrac{\dfrac{1}{n}\sum \limits_{i=1}^n (x_i - \bar x)^3}{[\dfrac{1}{n}\sum \limits_{i=1}^n (x_i - \bar x)^2]^{3/2}}S=[n1​i=1∑n​(xi​−xˉ)2]3/2n1​i=1∑n​(xi​−xˉ)3​
    • S值>0，就是正偏（右偏），表示均值>中位数>众数；<0表示负偏（左偏），表示均值<中位数<众数。

  • 峰态系数：数据分布集中强度的衡量。

  • 峰态系数公式：

    • K=1n∑i=1n(xi−xˉ)4[1n∑i=1n(xi−xˉ)2]2K = \dfrac{\dfrac{1}{n}\sum \limits_{i=1}^n (x_i - \bar x)^4}{[\dfrac{1}{n}\sum \limits_{i=1}^n (x_i - \bar x)^2]^{2}}K=[n1​i=1∑n​(xi​−xˉ)2]2n1​i=1∑n​(xi​−xˉ)4​
    • K值越大，顶峰越尖，越小，顶峰越平滑。

  • 重点：

    • 正态分布的峰态系数是3，如果特征计算出的峰态系数 K>5或K<1(和正态分布峰态系数之差大于2)，就说其不是正态分布了。
    • 最重要作用就是可以拒绝正态分布的假设(和正态分布峰态系数之差大于2就可以拒绝正态分布假设)

数据抽样

抽样代表了总体。

  • 主要分为：重复(有放回)和不重复(无放回)抽样。
  • 抽样方式：
    • 完全随机；
    • 等差距：按某个属性排序，等差距抽取；
    • 分类分层：根据各个类别的比例抽取。保证各个类别抽取的结果分布和原始一致；
  • 抽样平均误差计算公式：
    • 重复抽样：μx=σ2n\mu_x = \sqrt{\dfrac{\sigma^2}{n}}μx​=nσ2​​，σ2\sigma^2σ2是总体方差，nnn表示抽样数量。
    • 不重复抽样：μx=σ2n(N−nN−1)\mu_x = \sqrt{\dfrac{\sigma^2}{n}(\dfrac{N-n}{N-1})}μx​=nσ2​(N−1N−n​)​，σ2\sigma^2σ2是总体方差，nnn表示抽样数量，NNN总体样本数量。
  • 某些情况下，需要在一定误差之内，确定抽样数量。
    • 重复抽样：n=Zα/2δ2Δ2n=\dfrac{Z_{\alpha/2}\delta^2}{\Delta^2}n=Δ2Zα/2​δ2​，Zα/2Z_{\alpha/2}Zα/2​表示满足多少正确率所使用的倍数（比如正态分布满足95%正确率，这个值就是1.96，一般使用2），δ2\delta^2δ2表示总体方差，Δ2\Delta^2Δ2是我们要控制的方差(误差)。

数据分类

  • 统计学上把数据分为4类：
    • 定类（类别）：根据事物离散、无差别属性进行分类。优点：类别间无差距。
    • 定序（顺序）：可以界定数据的大小，但不能测定差值。有差距，大小不能确定。
    • 定距（间隔）：可以界定数据大小的同时，可测定差值，但无绝对零点。乘法除法比率是无意义的。
    • 定比（比率）：可以界定数据的大小，可测定差值，有绝对零点。

数据清洗

异常值处理

异常值包括：缺失值（空值null）、重复值、四分位数计算出来的超过上下界的值、超出常识的值。

识别：
（1）缺失值：Pandas的isnull()。
（2）重复值：Pandas的duplicated()。
（2）异常值：需要自己定义上下界，提取出非异常数值。Pandas的

处理：
（1）丢弃：缺失值用Pandas的drop()/dropna()。重复值用Pandas的drop()/drop_duplocated()。
（2）填充：离散：众数。连续：均值/中位数。也可以用边界值填充超过边界的异常值。Pandas的fillna()。插值填充（默认线性）Pandas的interpolate()–Series。

样本不平衡处理

所谓不平衡是指不同类别的样本量差异非常大，或者少数样本代表了业务关键数据，需要对少量样本的模式有很好的学习。
样本不平衡问题主要在分类问题上常见。

在工程上，应对样本均衡问题常从以下三方面入手：
1、数据相关处理：
（1）欠采样：在少量样本数量不影响模型训练的情况下，可以通过对多数样本欠采样，实现少数样本和多数样本的均衡。
（2）过采样：在少量样本数量不支撑模型训练的情况下，可以通过对少量样本过采样，实现少数样本和多数样本的均衡。
2、模型相关处理：
（1）模型算法：通过引入有倚重的模型算法，针对少量样本着重拟合，以提升对少量样本特征的学习。
3、评价指标相关处理
（1）设计相关指标，使指标偏向样本少的那类。

想了解更多数据相关方法，可以参考imbalanced-learn。

欠采样

欠采样，也叫下采样（under-sampling，us）。通过减少分类中多数类样本的数量来实现样本均衡。通过欠采样，在保留少量样本的同时，会丢失多数样本中的一些信息。经过欠采样，严格不能总量在减少。

欠采样有多种方法，常见的有：
1、随机删除
（1）分别确定样本集中多量样本数NmaxN_{max}Nmax​和少量样本数NminN_{min}Nmin​；
（2）确定采样样本集中多量样本与少量样本比值NratioN_{ratio}Nratio​；
（3）以少量样本为基准，确定多量样本采样总数Nsample=Nratio∗NminN_{sample}=N_{ratio}*N_{min}Nsample​=Nratio​∗Nmin​
（4）以NsampleN_{sample}Nsample​为限，对多量样本进行随机抽样。
缺点：不确定性很大，一般不采用这种方法。

算法中，可以以样本序号为种子，利用python自带的ranom.sample()函数进行行采样。

2、原型生成（prototype generation）
原型生成主要是在原有样本的基础上，生成新的子集样本来实现样本均衡，能保证样本分布和原先相同。
（1）以少量样本总数出发，确定均衡后多量样本总数NmaxN_{max}Nmax​；
（2）多量样本出发，利用K-means算法随机的计算K个多量样本的中心；
（3）认为K-means的中心点可以代表样本簇的特性，以该中心点s^\prime}代表样本簇；
（4）重复2、3两步NmaxN_{max}Nmax​次，生成新的多量样本集合S′=s1′,s2′,...,sNmax′S^\prime=s_1^\prime,s_2^\prime,...,s_{N_{max}}^\primeS′=s1′​,s2′​,...,sNmax​′​，且S′∉SS^\prime \notin SS′∈/​S。

下图所示，为三类共2000个样本点的集合，每个样本维度为2，label1个数为1861个，label2个数为108个，label3个数为31个。利用原型生成算法完成数据均衡后，样本整体分布没有变化。

3、原型选择（prototype selection）
NearMiss本质上是一种原型生成方法，即从多数类样本中选取最具代表性的样本用于训练，主要是为了缓解随机欠采样中的信息丢失问题。
NearMiss采用一些启发式的规则来选择样本，根据规则的不同可分为3类：
（1）NearMiss-1：选择到最近的K个少数类样本平均距离最近的多数类样本。
（2）NearMiss-2：选择到最远的K个少数类样本平均距离最近的多数类样本。
（3）NearMiss-3：对于每个少数类样本选择K个最近的多数类样本，目的是保证每个少数类样本都被多数类样本包围。

NearMiss-1和NearMiss-2的计算开销很大，因为需要计算每个多类别样本的K近邻点。另外，NearMiss-1易受离群点的影响，如下面第二幅图中合理的情况是处于边界附近的多数类样本会被选中，然而由于右下方一些少数类离群点的存在，其附近的多数类样本就被选择了。相比之下NearMiss-2和NearMiss-3不易产生这方面的问题。

过采样

过采样，也叫上采样（over-sampling）方法，通过增加分类中少数样本数量来实现样本均衡。缺点就是样本特征少时，会导致过拟合。

过采样有多种方法，常见的有：
1、随机重复
这个和前面讲的随机删除有异曲同工之妙，通过对少数样本随机复制，达到样本均衡。
最直接的方法是随机复制，但这样很容易过拟合。经过改进的抽样方法是通过在少数类中加入随机的噪声、干扰数据或通过一定规则产生新的合成样本。

下图所示，在进行复制前，Linear SVC只找到一个超平面，即认为样本集中仅有两类样本。随机复制后，Linear SVC找到了另外两个超平面。

2、样本构建
样本构建方法有很多，基础的有SMOTE（Synthetic Minority Oversampling TEchnique））。
SMOTE对少数类样本集中的每个样本xxx，从它在的少数类样本集中的K近邻中随机选一个样本yyy，然后在x,yx,yx,y的连线上随机选取一点作为新合成的样本。
SMOTE缺点：为每个少数类样本合成相同数量的新样本，可能会增大类间重叠度，并且会生成一些不能提供有益信息的样本。而且维度过高效果不好，所以实际不常用。

针对SMOTE缺点，衍生出SMOTEBoost、Borderline-SMOTE、Kmeans-SMOTE等。
SMOTEBoost：把SMOTE算法和Boost算法结合，在每一轮分类学习过程中增加对少数类的样本的权重，使得基学习器能过更好得关注到少数类样本。
Borderline-SMOTE：在构造样本时考虑少量样本周围的样本分布，选择少量样本集合（DANGER集合）————其邻居节点既有多量样本也有少量样本，且多量样本数不大于少量样本的点来构造新样本。
Kmeans-SMOTE：包括聚类、过滤和过采样三步。利用Kmeans算法完成聚类后，进行样本过滤，在每个样本簇内利用SMOTE算法构建新样本。

通过比较比较不同算法得到的样本构造，可得到如下结论：
不同算法构建的新样本在数量和分布上不同，其中利用SMOTE算法构建的新样本，没有考虑原始样本分布情况，构建的新样本受到“噪声”点的影响。同样ADASYN算法只考虑了分布密度而未考虑样本分布，构建的新样本也会受到“噪声”点的影响。Borderline-SMOTE算法考虑了样本的分布，构建的新样本能够比较好的避免“噪声”点的影响。Kmeans-SMOTE算法由于要去寻找簇后再构建新样本，可构建的新样本数量受限。
注意：“噪声”点对应类别上属于少量样本，但是分布上比较靠近边界或与多量样本混为一起。

模型相关

上面的过采样和欠采样都是从样本的层面去克服样本的不平衡。从算法层面来说，不同类型的错误造成的后果不同。

比如，医疗诊断中，把患者诊断为健康人、把健康人诊断为患者，这看起来都是“错了一次”，但是代价不同，前者可能会丧失拯救声明的机会，后者可能增加进一步检查的麻烦。
比如，门禁系统中，把可通行的人拦截在外，把陌生人放进门，前者使用户体验不佳，后者可能造成严重的安全事故。
比如，信用卡盗用检查中，把正常使用误认为盗用，把盗用误认为正常使用，前者使用户体验不佳，后者将使用户承受巨大损失。

为了权衡不同类型错误所造成的不同损失，可为错误赋予非均等代价。
常见的方法有：
（1）损失函数计算时，对不同类别的样本赋予不同的权重。使用Focal loss损失函数。
（2）使用模型集成方法，比如随机森林等。
（3）使用异常点检测算法。
（4）小样本学习。

1、Cost Sensitive算法
可根据任务设定一个代价矩阵，比如二分类任务，costijcost_{ij}costij​表示将第iii类样本预测为第jjj类样本的代价。一般来说，costii=0cost_{ii}=0costii​=0，若将第0类判别为第1类所造成的损失更大，则cost01>cost10cost_{01}>cost_{10}cost01​>cost10​，损失程度相差越大，cost01cost_{01}cost01​和cost10cost_{10}cost10​值的差别越大。

从学习模型出发，对某一具体学习方法的改造，使之能适应不平衡数据下的学习，研究者们针对不同的学习模型，如感知机、支持向量机、决策树、神经网络等分别提出了其代价敏感的版本。以代价敏感的决策树为例，可从三个方面对其进行改造以适应不平衡数据的学习：决策阈值的选择、分裂标准的选择、剪枝。这三个方面都可以将代价矩阵引入。

从贝叶斯风险理论出发，把代价敏感学习看成是分类结果的一种后处理，按照传统方法学习到一个模型，以实现损失最小为目标对结果进行调整。此方法优点在于它可以不依赖所用的具体分类器，但是缺点也很明显，它要求分类器输出值为概率。

从预处理的角度出发，将代价用于权重调整，使得分类器满足代价敏感的特性。

2、Meta Cost算法

（1）在训练集中多次采样，生成多个模型。
（2）根据多个模型，得到训练集中每条记录属于每个类别的概率。
（3）计算训练集中每条记录的属于每个类别的代价，根据最小代价，修改标签。
（4）训练修改过的数据集，得到新的模型。

3、Focal loss算法
Focal loss是在标准交叉熵损失基础上修改得到的，通过减少易分类样本的权重，使得模型在训练时更专注于难分类的样本。了解细节可参考Focal loss论文详解。

评价指标

样本不平衡的情况下，由于少量样本占比较小，如果仅考虑Error Rate或Accuracy，即使模型全部把少量样本分错，其整体的Error Rate和Accuracy还是比较高，因此，对于样本不平衡情况下，引入另外一个评价指标————G-mean。

G-mean：TPTP+FN×TNTN+FP\sqrt{\dfrac{TP}{TP+FN}\times\dfrac{TN}{TN+FP}}TP+FNTP​×TN+FPTN​​

在下一篇模型评估中，详细讲解了评价指标。

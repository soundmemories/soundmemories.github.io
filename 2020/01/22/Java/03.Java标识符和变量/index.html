Java标识符和变量

发表于 2020-01-22 分类于 Java 本文字数： 702 阅读时长 ≈ 3 分钟

标识符

  • 标识符的命名规则（必须遵守，否则编译或运行错误）
    • Java的标识符必须用26个英文字母的大小写、数字0-9，美元符号$，下划线_。
    • Java的标识符不能使用关键字（包含保留字）和特殊值。
    • 不能用的有50个关键字+3个特殊值。
    • 不能使用空格。
    • 数字不能开头。
    • Java的标识符严格区分大小写。
  • 标识符的命名规范（建议遵守，不遵守会被鄙视）
    • 见名知意：别人一看就知道这个类、变量、方法的意义在哪里。
    • 类名、接口名等：要求每一个单词的首字母大写。
    • 变量名、方法名等：要求从第二个单词开始，首字母大写，其余字母小写。
    • 包名等：要求所有单词都小写，每个单词之间使用 . 分割。
    • 常量名：要求所有单词都大写，每个单词之间使用 _ 分割。

变量

作用：用来存储值（数据）。
在Java中，变量本质上，就是代表一块内存区域。
变量中的数据是存储在JVM内存中。

  • 变量有三个要素：
    • 数据类型：字符串、小数、整数、单个字符、布尔值、对象..等。
    • 变量名：在程序中，通过变量名来进行访问这块内存及其里面的数据。
    • 值：变量中存的具体的值，这个值是可以变的。

Define：定义，声明。
Use：使用。
Variable：变量。

  • 变量使用的原则：
    • 先声明后使用，包括后面用到的类、方法等，都是遵循先声明后使用。
    • 变量必须先初始化（第一次赋值）后再访问它的值（打印、计算、比较操作）。
    • 变量的声明，在同一个作用域中，只能声明一次，但是赋值可以很多次。作用域：从变量声明处开始，到它所属的{}结束。
  • 变量如何声明？
    • 格式：数据类型 变量名;
  • 变量如何赋值？
    • 格式：变量名 = 变量值;

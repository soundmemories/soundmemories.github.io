Java运算符优先级

发表于 2020-01-31 分类于 Java
本文字数： 313 阅读时长 ≈ 1 分钟

运算符优先级

  • 优先级由高到低：
    • . ()
    • ++ -- ~ !
    • * / %
    • + -
    • << >> >>>
    • < > <= >= insatanceof
    • == !=
    • &
    • ^
    • |
    • &&
    • ||
    • ? :
    • = *= /= %=
    • += -= <<= >>=
    • >>>= &= ^= |=
  • 通俗记忆：
    • .（面向对象用）和()
    • 自增，自减，按位取反，非
    • 算术运算符
    • 左移右移的位运算符
    • 比较运算符
    • || && | ^ &
    • 条件运算符
    • 赋值类运算符

要准确的记忆每一种运算符的优先级是困难的。
我们遵循代码基本原则：1）表达式不要写太复杂，可以分为多行；2）如果非要混合运算，那么先算的用()括起来。

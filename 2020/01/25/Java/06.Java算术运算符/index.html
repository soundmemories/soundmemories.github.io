++i和i++区别

  • 自增：++
    • 对于自增变量本身来说，都会+1；
    • ++在前，先自增，然后取自增后变量的值；
    • ++在后，先取变量的值，然后变量自增；
    • 但是不管怎么样，自增变量的取值与自增操作一前一后一定是一起完成的。
  • 自减：--
    • 类同自增

Java算术运算符

发表于 2020-01-25 分类于 Java
本文字数： 4.2k 阅读时长 ≈ 4 分钟

算术运算符

加：+
减：-
乘：*
除：/（整数/整数，结果只保留整数部分）
取模（取余）：%（被模数%模数，只看被模数的正负号）。

++i和i++区别

  • 自增：++
    • 对于自增变量本身来说，都会+1；
    • ++在前，先自增，然后取自增后变量的值；
    • ++在后，先取变量的值，然后变量自增；
    • 但是不管怎么样，自增变量的取值与自增操作一前一后一定是一起完成的。
  • 自减：--
    • 类同自增

运算符优先级

要准确的记忆每一种运算符的优先级是困难的。
我们遵循代码基本原则：1）表达式不要写太复杂，可以分为多行；2）如果非要混合运算，那么先算的用()括起来。

算术运算符

++i和i++区别

  • 自增：++
    • 对于自增变量本身来说，都会+1；
    • ++在前，先自增，然后取自增后变量的值；
    • ++在后，先取变量的值，然后变量自增；
    • 但是不管怎么样，自增变量的取值与自增操作一前一后一定是一起完成的。
  • 自减：--
    • 类同自增

赋值运算符

  • 赋值运算符=
    • 永远是把其右边的常量值、变量中值、表达式计算的值赋值给其左边的变量，其左边只能是一个变量。
    • 运算的顺序：把右边的整个表达式先算完，才会做最后的赋值操作。
  • 扩展的赋值运算符
    • +=、-=、*=、/=、%=、...。

比较运算符

比较运算符：
1.大于：>
2.小于：<
3.大于等于：>=
4.小于等于：<=
5.等于：==（谨防与赋值的=混淆）
6.不等于：!=

比较运算符，计算完后的结果只有两种：true，false。
比较运算符的表达式，可以作为：（1）判断的条件（2）逻辑运算符的操作数。

比较运算符能够用于基本数据类型，不能用于引用数据类型。
除了 == 和 != ，可以引用数据类型的比较，用于比较对象地址。
如果想比较引用数据类型的值，使用.equals()。

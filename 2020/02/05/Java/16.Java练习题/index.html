选择题

short s = 1;要给s加上2，可以如何实现？
A：s = s + 2;
B：s += 2;
C：s = (short)(s + 2);
D：s = (short)s + 2;
答案：C

如下代码的计算结果是：
int i = 1;
i *= 0.2;
i++;
System.out.println(“i=” + i)
答案：i=1（最后赋值的i是int类型，int类型只保留整数部分）

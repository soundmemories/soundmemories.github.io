条件判断

单分支条件判断

if()中表达式必须是boolean类型的结果：1）boolean类型的值；2）比较表达式；3）逻辑表达式。
当{}中的语句只有一句时，可以省略{}，不建议省略。

双分支条件判断

if()中表达式必须是boolean类型的结果：1）boolean类型的值；2）比较表达式；3）逻辑表达式。
当{}中的语句只有一句时，可以省略{}，不建议省略。

多分支条件判断

if()中表达式必须是boolean类型的结果：1）boolean类型的值；2）比较表达式；3）逻辑表达式。
当{}中的语句只有一句时，可以省略{}，不建议省略。

总结：
不管有几个分支，最终只会执行其中的一个；
这么多条件表达式是顺序往下判断的；
如果上面有条件成立了，下面的就不看了；
如果上面的条件不成立，才会去看下面条件。
当多个条件的范围是“互斥”（没有重叠部分），顺序可以随意；
当多个条件的范围是“包含”（有重叠部分），顺序不能随意，小的在上，大的在下。


#### 嵌套条件判断

当嵌套时，里面的条件判断要被执行的前提，外面的if或else要先满足。

练习题（random）

随机数产生：


假设你想开发一个玩彩票的游戏：
程序随机地产生一个两位数的彩票，提示用户输入一个两位数，然后按照下面的规则判定用户是否能赢？
1)如果用户输入的数匹配彩票的实际顺序，奖金10 000美元。
2)如果用户输入的所有数字匹配彩票的所有数字，但顺序不一致，奖金 3 000美元。
3)如果用户输入的一个数字仅满足顺序情况下匹配彩票的一个数字，奖金1 000美元。
4)如果用户输入的一个数字仅满足非顺序情况下匹配彩票的一个数字，奖金500美元。
5)如果用户输入的数字没有匹配任何一个数字，则彩票作废。

两位数的范围：[10,99]
Math.random() 产生[0,1)范围的随机数
Math.random() * 90：得到[0,90)
Math.random() * 90 + 10：得到[10,100)，即得到 [10,99]
简化：(int)(Math.random() * 90 + 10)产生一个两位数的随机数。

选择结构

case后面必须是常量值；
case后面的常量值不能重复；
switch()中的表达式的类型有要求：
1）允许4种基本数据类型：byte,short,int,char；
2）允许2种引用数据类型：枚举（JDK1.5之后）、String（JDK1.7之后）。

入口：
1）当switch()中的表达式的值与某一个case后面的常量值“相等”，就会从这个case进入；
2）当switch()中的表达式的值与所有的case后面的常量值都“不相等”，就会寻找default进入。
出口：
1）自然出口：switch的结束；
2）中断出口：break（或return等）。
一旦找到“入口”就会顺序往下执行，直到遇到“出口”。
default的位置不一定在最后，习惯上我们写在最后。

循环结构

重复执行某些代码时，需要循环结构。
循环结构：
（1）for循环；
（2）while循环；
（3）do...while循环；
（4）foreach增强循环。

for循环

死循环，无限循环。
for(;;)其中的两个 ; 不能多也不能少。


循环条件表达式1要么不写，要么必须是boolean类型。
执行的过程：
（1）执行循环变量初始化表达式1;
（2）进行循环条件表达式2判断；
（3）如果循环条件表达式2成立，执行循环体语句4，然后再执行循环变量迭代表达式3。
（4）完事后回到（2），如果循环条件表达式2不成立，结束整个for循环；成立回到（3），直到循环条件表达式2不成立。

练习题：
  • 100sum
  • 100even
  • 1-150

while循环

执行过程：
（1）先判断循环条件；
（2）如果循环条件成立，那么就执行循环体语句，再回到（1）；不成立，那么就结束while循环。

仔细分析，仍然有四个要素：
（1）循环变量的初始化，例如：int i = 1;
（2）循环条件，例如：i<=100；
（3）循环体语句，例如： sum+=i;
（4）迭代语句，例如：i++。

结论：
（1）各种循环之间是可以互相替换的，只不过，在某些场景，某一种循环更合适（可读性方面）
（2）for循环一般用于：次数比较明显，以及数组等遍历。
（3）while循环一般用于：条件比较明显，次数不太明显。


练习题：
  • narcissus1
  • narcissus2

do...while循环

执行过程：
（1）先执行一次循环体语句块；
（2）然后再判断条件，看是否继续下一次；
（3）如果条件成立，再次执行循环体语句块；然后回到（2）；如果条件不成立，直接结束do...while。

结尾的 ; 要注意。
do...while循环与for和while的最大区别：它的循环体语句块是至少执行一次。

  • DoWhile
  • DoWhile_For
  • DoWhile_While

break和continue

break：
（1）用于switch..case，结束switch。
（2）用于循环结构，结束循环；
（3）如果break在内循环中，结束的内循环；
（4）如果break在内循环外，外循环内，结束的是外循环。
（5）如果要用break结束某个外循环，可以给此外循环取一个标签名。
结束一个循环有两种情况：1）循环条件不满足；2）遇到了break等中断。

  • Break1
  • Break2
  • Break3

continue：提前结束本次循环，继续下一次循环；只能用于循环结构。

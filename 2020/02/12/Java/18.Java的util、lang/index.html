Java的util、lang

发表于 2020-02-12 分类于 Java
本文字数： 13k 阅读时长 ≈ 12 分钟

java.util

封装了很多常用工具类。

Arrays

  • public static int binarySearch(int[] a,int key)
    • 使用二分搜索法来搜索指定的 int 型数组，以获得指定的值。
    • 要求数组是有序的，否则结果不确定。
    • 如果key在a数组中存在，就返回找到的第一个的下标，如果不存在返回负数（-(插入点)-1），插入点是值应在的位置。
  • public static int[] copyOf(int[] original,int newLength)
    • 从original原数组中复制得到一个新数组，新数组的长度为newLength。
  • public static int[] copyOfRange(int[] original,int from,int to)
    • 从original原数组中把[from,to)复制出来，构成新数组。
    • 要求from必须在original的合理下标范围内，to可以超过长度。
  • public static boolean equals(int[] a,int[] a2)
    • 比较两个数组的长度和内容是否一致。
  • public static void fill(int[] a,int val)
    • 把a数组的元素中val填充。
  • public static void sort(int[] a)
    • 把数组a按照升序排列。
  • public static String toString(int[] a)
    • 把a数组的元素，遍历，拼接为一个字符串返回，效果[1,3,5]。

和数组相关的常用方法（java.lang.System类）：

  • public static native void arraycopy(Object src, int srcPos,Object dest, int destPos,int length)
    • 第一个参数：src 原数组对象。
    • 第二个参数：srcPos 从原数组的[srcPos]下标开始复制。
    • 第三个参数：dest 目标数组对象。
    • 第四个参数：destPos 目标数组从[destPos]开始存储。
    • 第五个参数：length 表示从原数组复制几个元素。
    • 说明：如果src和dest是同一个数组的话，那么就会实现数组的元素的移动效果。
      • srcPos > destPos ：往左移动，一般用于删除。
      • srcPos < destPos ：往右移动，一般用于插入。

Object：是所有引用数据类型的根父类。
那么根据多态规则，Object类型的变量、形参，就可以接收任意的引用数据类型的对象，包括数组。

Arrays的sort方法有两种：
（1）void sort(Object[] arr)：根据元素的自然顺序对指定对象数组按升序进行排序。数组中的所有元素都必须实现 Comparable 接口。
（2）void sort(Object[] arr, Comparator c)：根据“指定比较器”产生的顺序对指定对象数组进行排序。数组中的所有元素都必须是通过“指定比较器”可相互比较的。
以上排序，默认返回正数的排序在最后，也就是从小到大排序，想从大到小排序，把返回正数的改成负数即可。

Comparator

  • java.util.Comparator<T>
    • int compare(T o1, T o2)：
      • 代表Java中比较两个对象的大小标准。而且是一种“定制”比较的标准。
      • 因为<T>是泛型，我们还没讲。现在就把T看成Object。
      • 这个接口中没有规定如何比较两个对象的大小，但是规定了：
        • 如果你认为o1 > o2，就返回正整数表示。
        • 如果你认为o1 < o2，就返回负整数表示。
        • 如果你认为o1 = o2，就返回0表示。
          Java中还有另一个java.lang.Comparable，这是一种“自然比较”的标准。

java.lang

它提供了java中的基础类。

Comparable

  • java.lang.Comparable
    • int compareTo(Object obj)：
      • 代表Java中比较两个对象的大小标准。而且是一种“自然”比较的标准。
      • 因为<T>是泛型，我们还没讲。现在就把T看成Object。
      • 这个接口中规定如何比较两个对象的大小：
        • this与obj对象比较，this > obj，返回正整数。
        • this与obj对象比较，this < obj，返回负整数。
        • this与obj对象比较，this = obj，返回0。

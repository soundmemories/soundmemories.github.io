项目一：家庭记账软件

模拟实现基于文本界面的《家庭记账软件》。
能够记录家庭的收入、支出，并能够打印收支明细表。
项目采用分级菜单方式。主菜单如下：

假设家庭起始的生活基本金为10000元。
每次登记收入（菜单2）后，收入的金额应累加到基本金上，并记录本次收入明细，以便后续的查询。
每次登记支出（菜单3）后，支出的金额应从基本金中扣除，并记录本次支出明细，以便后续的查询。
查询收支明细（菜单1）时，将显示所有的收入、支出明细列表。

明细表格的对齐，可以使用制表符“\t”来实现。

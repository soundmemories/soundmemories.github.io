heapq

该模块提供了堆排序算法的实现。堆是二叉树，最大堆中父节点大于或等于两个子节点，最小堆父节点小于或等于两个子节点。

heapq有两种方式创建堆：

  1. 使用一个空列表，然后使用heapq.heappush()函数把值加入堆中。
  2. 使用heap.heapify(list)转换列表成为堆结构。

heapq.merge(*iterables)：用于合并多个排序后的序列成一个排序后的序列，返回排序后的值的迭代器。类似于sorted(itertools.chain(*iterables))，但返回的是可迭代的。

堆创建好后，可以通过heapq.heappush(heap, item)函数添加元素。


可以通过比较添加元素，heapq.heappushpop(list, item)。
判断item与堆的第一个元素值对比；如果item大，则堆删除并返回第一个元素，然后添加新元素值item，如果item小，则返回item，原堆不变。

可以通过heapq.heappop()函数弹出堆中最小值。
如果需要删除堆中最小元素并加入一个元素，可以使用heapq.heaprepalce()函数。


获取堆中最大或最小的范围值，则可以使用heapq.nlargest()或heapq.nsmallest()函数。
这两个函数还接受一个key参数，用于dict或其他数据结构类型使用。

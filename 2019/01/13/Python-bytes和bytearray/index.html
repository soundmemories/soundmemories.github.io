Python-bytes和bytearray

1.1 bytes和bytearray

Python3引入两个新类型，bytes和bytearray都是字节序列
bytes：不可变字节序列。
bytearray：字节数组、可变。

字符串与bytes：
  可字符串是字符组成的有序序列，字符可以使用编码来理解
  bytes是字节组成的有序的不可变序列
  bytearray是字节组成的有序的可变序列

1.2 编码和解码

编码与解码：
  字符串按照不同的字符集编码encode返回字节序列bytes

十进制：09（4857）、AZ（6590）、az（97122）

汉字编码：utf-8：3个字节1个汉字；gbk：2个字节1个汉字；

字典遍历和移除：

python2中，上面方法返回一个新的列表，占据新的内存空间。所以2中建议使用iterkeys、itervalues、iterterms返回一个迭代器，而不是一个copy。

python3中，keys()、values()、items()（kv对的2元组）方法返回一个类似生成器的可迭代对象，不会把函数的返回结果赋值到内存中。

  • 字典对象，可以使用len()、iter()、in()操作。
  • keys()、items()：可以当set用，比如集合运算，前提是所有values可hash。
  • 不能边view边增删它。可以先把key拿出来放到可迭代对象中再操作。

注意用for边循环自己，边删除自己时，不能从头删（删除当前后面会前进，会乱掉），要从尾部删除！

有序字典：OrderedDict

collections.OrderedDict([items])

  • 字典无序，可以使用OrderedDict记录插入顺序，3.6版本字典就是记录key插入顺序，但2.x版本不是。建议使用这种方式记录顺序！

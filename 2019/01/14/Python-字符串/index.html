Python-字符串、常用命令

发表于 2019-01-14 分类于 Python
本文字数： 4.9k 阅读时长 ≈ 4 分钟

字符串：

  • 单引号、双引号、三引号都可以。
  • 特性：
    • 一旦定义不能改变（赋值），所以生成的都是新字符串。
    • python中只有字符串类型，没有字符类型。
    • 本身可迭代。

使用str()时注意：

把range(5)当字符串了！

使用print(sep,end)时注意：

sep：输出多个值之间的分隔符 end：结束时的分隔符。

三元运算符：

for _ in range(10): 当计数器时可以不写变量，使用_代替。

其他函数：

"string".join(iterable)->str：使用string作为分隔符，iterable是可迭代字符串。iterable若不是字符串可用map(str,iterable)转换成str。

.replace(old, new[,count]))->str:old旧字符，new新字符，count替换次数。

.strip([chars])->str:默认去除字符串两头的空白字符，()里面指定字符时，\t\r\n之类的就算非空白字符了。非整体匹配，挨个匹配，两边匹配到第一个不是指定的字符就停止。lstrip,rstrip

.index(sub[,start[,end]])->index:前闭后开查到返回索引号，找不到返回异常。O(n)

.find(sub[,start[,end]])->index：找不到返回-1
.rfind()->index:右到左搜索返回找到的索引号。

'v' in 字符串名：在字符串中查找字符'v'返回bool值。

.count(sub[,start[,end]])->int:返回列表中匹配value的次数，O(n)

.startswith(prefix[,start[,end]])->bool：头匹配O(1)
.endswith(suffix[,start[,end]])->bool：结尾匹配O(1)

d  = 'ab'*5：'ababababab'返回的是新字符串
d += 'c'：'abababababc'（字符串+=特殊，正常+=是就地修改）返回的是新字符串，相当于d=d+'c'生成新的字符串在赋值给d，字符串不可修改，原字符串是被清空了。

.split(sep=切割符,maxsplit=最大切割数)->list：默认空白为切割符、切掉尽可能多的空白，切割符丢掉。
.splitlines()：按行切，''.splitlines(True)：保留切掉的东西
'a  b'.split(' '):['a','','b']：把b前面2个空格，都切除，但是切除后2个空格之间是空，所以输出''。三个空格：切割3个空格输出2个''['a', '', '', '\nb']
'a  b'.rsplit(' '):['a','','b']

.partition(sep)->tuple:必须要有分隔符
'a,b,c'.partition(','):('a', ',', 'b,c')返回三元组，中间元素是后面()里的切割符
示例：'a,b,c'.rpartition(','):('a,b', ',', 'c') 和 'a,b,c'.rpartition(' '):('','','a,b,c')

enumerate(iterable, start=0):枚举，生成(下标、元素)的2元组，可立即使用解构。

字符串格式化format：

.format(*args, **kwargs)->str
"{{}}".format()：打印花括号    
'{:.2f}'.format(1.2345)：'1.23' 控制小数点宽度。
'{:.2}'.format(12.2345)：'1.2e+01'不写f变成科学计数法。
'{:.3%}'.format(1/3)：'33.333%' 百分制输出。

'{:2}'.format(123)：'123'超出范围全都输出。
'{0:3}'.format(2)：0索引号，3表示占三位，默认右对齐，
'{:0<3}'.format(2)：左对齐,空位补0
'{:>3}'.format(2)：右对齐
'{:^3}'.format(2)：居中
'{:#x}'.format(127)：转化成16进制'0x7f'，#输出加0x。
'{}'.format(*"192.168.0.1".split('.'))：'192'，*号能把后面一个整体的元素按顺序输出直到{}填满。

示例：
‘a{}’.format(1)：‘a1’
‘a{}’.format(1,2)：‘a1’
‘a{0} {1}’.format(1,2)：’a1 2’
‘{} {}’.format(‘a’,[‘b’])：“a [‘b’]”
“{} {}”.format(p1.x,p1.y)：p1是namedtuple可以用.来引用它的属性。

字符串bytes和bytearray

  • 字符串是字符组成的有序序列，字符可以使用编码来理解。

  • bytes是字节组成的有序的不可变序列。

  • bytearray是字节组成的有序的可变序列。

  • 编码与解码

    • 字符串按照不同的字符集编码encode返回字节序列bytes
    • encode(encoding='utf-8', errors='strict') -> bytes
    • 字节序列按照不同的字符集解码decode返回字符串
    • bytes.decode(encoding="utf-8", errors="strict") -> str
    • bytearray.decode(encoding="utf-8", errors="strict") -> str

  • 定义

    • 使用b前缀定义
    • 只允许基本ASCII使用字符形式b’abc9’
    • 使用16进制表示b"\x41\x61"
    • bytes() 空bytes
    • bytes(int) 指定字节的bytes，被0填充
    • bytes(iterable_of_ints) -> bytes [0,255]的int组成的可迭代对象
    • bytes(string, encoding[, errors]) -> bytes 等价于string.encode()
    • bytes(bytes_or_buffer) -> immutable copy of bytes_or_buffer 从一个字节序列或者buffer复制出
      一个新的不可变的bytes对象

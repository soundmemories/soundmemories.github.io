Python-Bytes和Bytearray

发表于 2019-01-18 分类于 Python
本文字数： 814 阅读时长 ≈ 1 分钟

Bytes和Bytearray

Python3引入两个新类型，bytes和bytearray都是字节序列：

  • bytes：不可变字节序列。
  • bytearray：字节数组、可变。

字符串与bytes：

  • 可字符串是字符组成的有序序列，字符可以使用编码来理解
  • bytes是字节组成的有序的不可变序列
  • bytearray是字节组成的有序的可变序列

编码和解码

编码：

  • 字符串按照不同的字符集编码（encode）返回字节序列bytes。 解码：
  • 字节序列按照不同的字符集解码（decode）返回字符串。

编码：字符->bytes
解码：bytes/bytearray->字符

英文编码：
ASCII码表是指对字符的编码：

十进制：

  • 0~9（48~57）
  • A~Z（65~90）
  • a~z（97~122）

汉字编码：

  • utf-8：3个字节1个汉字。
  • gbk：2个字节1个汉字。

    bytes

bytes 函数返回一个新的 bytes 对象，该对象是一个 0 <= x < 256 区间内的整数不可变序列。它是 bytearray 的不可变版本。

  • bytes是字节组成的有序的不可变序列。
  • bytearray是字节组成的有序的可变序列。

    字符串是字符组成的有序序列，字符可以使用编码来理解。

bytes([source[, encoding[, errors]]])


参数总结：

  • 如果没有输入任何参数，默认就是初始化数组为0个元素；
  • 如果 source 为整数，则返回一个长度为 source 的初始化数组；
  • 如果 source 为可迭代类型，则元素必须为[0 ,255] 中的整数；
  • 如果 source 为字符串，则按照指定的 encoding 将字符串转换为字节序列；
  • 如果 source 为与 buffer 接口一致的对象，则此对象也可以被用于初始化 bytearray。

bytearray

bytearray() 方法返回一个新字节数组。这个数组里的元素是 可变的，并且每个元素的值范围: 0 <= x < 256。

bytearray([source[, encoding[, errors]]])
参数同bytes。

字体相关

属性名 作用 属性值
font-size 字体大小 长度单位：px、em、%
默认值是16px，最小值是12px。
注意！行内元素之间的换行符，会被解释为空格，它也会随着设置变化，从而影响布局。
font-weight 字体粗细 normal：正常。
lighter：细。
bold：粗。
数字形式（100-900）：100-300细体，400-500正常，600以及以上粗体。
font-style 斜体字 normal：正常。
italic：斜体（字体设计的斜体）。
oblique：强制变斜体（不管字体设没设计斜体，都变斜体）。
font-family 字体族科。
备注：电脑上必须安装该字体。		 字体名称：""可加，也可以不加。
字体列表：列出一系列字体，按顺序能用哪个用哪个。
衬线字体和非衬线字体：
serif：表示衬线字体，笔画粗细不一致，如宋体、仿宋体、楷体等。
sans-serif：表示非衬线字体，笔画粗细一致，如微软雅黑、黑体、Helvetica。
font 设置多种字体样式。
备注：font是复合属性，像上面font-size等是其子属性。		 多个值，使用空格分隔（注意复合属性不要和子属性一起写，容易混乱）。

示例：

  • font-size
  • font-weight
  • font-family
  • font

文本相关

属性名 作用 属性值
letter-spacing 字间距 长度单位：px、em、%
word-spacing 词间距（中文无效果） 长度单位：px、em、%
text-decoration 文本修饰线 none：无修饰线。
underline：下划线。
overline：上划线。
line-throuth：删除线
text-indent 首行缩进（推荐em） 长度单位：px、em、%
text-align 元素中的文本在当前元素内的水平对齐方式。
注意！不是元素相对页面的对齐方式。		 left：左对齐。
right：右对齐。
center：居中对齐
vertical-align 1.设置行内或行内块元素与同行文本纵向对齐方式。
2.设置上标字和下标字。
3.设置单元格中内容的纵向对齐方式，只能用于 td、th。		 baseline：基线对齐。
top：顶线对齐。
middle：中线对齐。
bottom：底线对齐。
sub：下标字。
super：上标字。
长度：元素底部与文字基线的距离。
line-height 设置行高（上行文字中线与下行文字中线之间距离）
注意！区分行距（上行文字底线和下行文字顶线之间距离）。
注意！行距不能设置，但调整行高大小，行距会受影响。
备注：第一行文字中线与元素顶部距离是行高一半，最后一行文字中线与元素底部距离是行高一半！当只有一行文字，那么设置行高与高度一致，可实现文字垂直居中。
备注：line-height是font的子属性，他俩一起使用时注意覆盖问题，可以将行高写在font复合属性中。
备注：由于字体设计的原因，文字在一行中，并不是绝对垂直居中，若一行中都是文字，不会太影响观感。		 normal：由浏览器根据文字大小决定的一个默认值。
px：像素。
em：参考自身font-size的倍数。常用1.5~2倍之间。
%：参考自身font-size的百分比。
备注：行高过小：导致文字重叠，且最小值是0，不能为负数（负数会无效，默认为normal）。
备注：行高是可以继承的。
  • text-decoration
  • text-indent
  • text-align/vertical-align
  • line-height

背景相关

属性名 作用 属性值
background-color 背景颜色。
注意！元素默认背景颜色是透明，跟随父元素。		 颜色设置（默认值是transparent（透明））
background-image 设置背景图像地址。
备注：图片大小<元素宽高的框大小，会自动重复图片显示，需配合设置background-repeat参数。
备注： 图片大小>元素宽高的框大小，只会显示图片一部分，原点为左上角。		 url(地址)
background-repeat 设置背景图像在元素内重复方式。 repeat：重复。
repeat-x：横向重复。
repeat-y：纵向重复。
no-repeat：不重复。
background-position 设置背景图像元素内位置，有两个子属性。
background-position-x：设置x位置。
background-position-y ：设置y位置。
备注：background-repeat如果非no-repeat，不管设置位置如何，也会填满整个元素。		 关键字。
长度坐标。
百分比。
background-attachment 背景图像固定。
背景图像只能显示图像与元素位置重合的位置。		 scroll：随元素滚动，默认值。
fixed：固定，固定的是窗口原点对图片原点。
background 背景复合属性，非常优秀的复合属性。 前面列出的属性都可以使用，多个值使用空格分隔，没有数量、顺序要求。
  • background-color
  • background-repeat
  • background-position
  • background-attachment
  • background

列表相关

属性名 作用 属性值
list-style-type 设置列表项图标。 none：无，常用值。
disc：实心圆，默认值。
circle：空心圆。
square：实心方块。
...
list-style-position 设置列表项图标位置。 outside：图标在li外面，默认值。
inside：图标在li里面。

表格相关

属性名 作用 属性值
table-layout 设置列宽固定。 auto：根据内容自动变化，默认值。
fixed：固定，均分列宽。
border-spacing 设置单元格之间的距离。 长度。
border-collapse 合并单元格边框，相邻单元格共用一个边框。 separate：分离的，默认值。
collapse：合并，设置此值后border-spacing失效。
caption-side 标题位置。 top：表格上面。
bottom：表格下面。
empty-cells 没有内容的单元格显示还是隐藏。 show：显示，默认值。
hide：隐藏，只有border-collapse为separate才能看出效果。

注意： 表格相关的属性只能设置到 table 标签上才生效！

盒子的边框

CSS 属性名 功能 属性值
border-style 边框风格。 none：无风格，默认值。
solid：实线。
dashed：虚线。
dotted：点线。
double：双实线。
...
border-color 边框颜色。 颜色，默认值是文字颜色。
border-width 边框宽度。 长度，默认值是3px。
border 同时设置风格、颜色、宽度。
border的子属性：border-style、border-color、border-width。
border-style的子属性：border-left-style、border-right-style、border-top-style、border-bottom-style。
border-color、border-width子属性同理。		 多个值使用空格分隔。
border-left/right/top/bottom-style
border-left/right/top/bottom-color
border-left/right/top/bottom-width
border-left/right/top/bottom
border-left的子属性：bordre-left-style、border-left-color、border-left-width。
border-right、border-top、border-bottom子属性同理。

1.如果四周要设置的颜色、风格、宽度一样，使用border。
2.如果四周要设置的颜色、风格、宽度不一样，使用border-left/right/top/bottom。
